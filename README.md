# Contagem-de-Objetos-PBM-ASCII-Processamento-de-Imagens
Algoritmo que informa o número total de objetos na imagem, e quantos objetos com furos e sem furos a imagem contém. A entrada do algoritmo é uma imagem binária no formato PBM ASCII (PGM tipo P1). O sistema deve funciona no modo console, recebendo a imagem como entrada na linha de comando e imprimindo os dados obtidos.
